REALIST® VALUEMAP™

Realist® ValueMap™ from CoreLogic is the standard for current and accurate property valuations. It is a map-based search tool that provides more than 140 million estimated home valuations and detailed property characteristics in an easy-to-understand format.

Sandicor Subscribers have the option to incorporate Realist® ValueMap™ into their personal website by installing the ValueMap™ widget, thus providing visitors access to an automated home valuation service that reflects public record sales enhanced by MLS data! This valuable functionality is currently available as an option in Classic Realist®; however, it will be moving to the new Realist® on Tuesday, October 23. Please note: Beginning October 23, all Sandicor Subscribers using the ValueMap™ widget will need to access this product within the New REALIST® product.
