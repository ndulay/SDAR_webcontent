DAY 1

Custom Searching/Map Searching

Tuesday, October 8
9:15 a.m. – 11:15 a.m.
This course will provide you with the skills to effectively and efficiently create custom property searches as well as map searches. Course topics include printing and email reports as well as setting your Tempo Personal Defaults.

Sandicor Mobile

Tuesday, October 8
11:30 a.m. – 12:30 p.m.

This course will teach you how to access Sandicor’s MLS data via your cell phone. You will learn how to use the specific iPhone/iPad, BlackBerry or Android apps, or access the browser-based application from any cell phone with Internet access.

Prospecting with Client Gateway

Tuesday, October 8
1:30 p.m. – 3:15 p.m.

Client Gateway is a Tempo 5 feature that allows an agent to create customized websites using a prospect’s search criteria. This course is designed to assist an agent to create custom searches, which are the foundation of Client Gateway. Course topics include creating Prospect Defaults and the Client Gateway website, as well as how to monitor the Client Gateway on an ongoing basis.

Add/Edit Listings

Tuesday, October 8
3:30 p.m. – 4:30 p.m.

In this course you’ll learn how to easily add or edit your listings, as well as how to quickly view your listing inventory information.

 

DAY 2

Fusion – Searching/Map Searching

Tuesday, October 15
9:15 a.m. – 11:00 a.m.

This course will introduce you to the next generation of Sandicor’s MLS software, Fusion MLS™, available as an alternative to the existing TEMPO™ software, which will remain available. Designed to be platform/browser independent, Fusion MLS™ can be used on MACs, as well as Chrome, Firefox, and Safari browsers. This course will teach you how to navigate the Fusion MLS™ system and use the interactive search and map search options. You’ll learn how to save time by manipulating a search map’s boundaries without starting over, and how to save listings of interest in your personalized folder for later action.

Fusion - Comparable Market Analysis

Tuesday, October 15
11:15 p.m. – 12:30 p.m.

This course will teach you how to create a Comparative Market Analysis (CMA), which is used to help sellers determine the value of their home. You will learn how to create a Quick CMA, plus go through the steps of the CMA Wizard to create a multiple CMA, including a Net Sheet.

Fusion – Contact Manager & Prospects

Tuesday, October 15
1:30 p.m. – 2:30 p.m.

This course will teach you how to use the Contact Manager and auto-notification in Fusion MLS™, an alternative to the existing TEMPO™ software. You will learn how to use the Contact Manager system to manage your client’s personal information as well as their property choices.

Realist Tax

Tuesday, October 15
2:45 p.m. – 4:30 p.m.

This course will teach you how to use Realist Tax within Tempo to obtain tax information for a specific property and how to export that data for marketing purposes. Course topics include Realist tax searches, farming, viewing assessor parcel maps, and printing labels. You will also learn how to use the Realist map search and how to do a CMA using tax data.
